Hi,

This is my first cut at getting freevo working on RH 7.3 with a proper
SPEC file.

The spec file builds only the x11 and sdl OSDs, as I don't have a Matrox
G400 card. (The build process is a little broken for the Matrox as well).
It stores the binaries in /usr/local/freevo as per the Makefile. I haven't
tried to make it FHB compliant yet as I plan to track the Makefile for
installation targets (ie., it'll happen when the Makefile is FHB
complaint).

It works with python2 from the RH 7.3 distro, some packages (PyXML) need
to be rebuilt for python2, which I've named PyXML2 to allow both to
coexist on my system.

PIL doesn't come with a spec file. I've created two versions, one for
the default python 1.5 release, and the other for python2. PIL2 is
the one that's really needed for freevo.

You can put them up on the freevo website for downloading if it is useful.
No guarantee that it'll work for anyone else though.

I'm still trying to get freevo working properly on my system, I
managed to get the x11 simulated menu to display but none of the actual
programs work yet. Unfortunately, I'm not familiar with Python or X
programming, so I can only take a stab at most of the freevo related
issues.
